# Bash: Current date and time

* Get current date:

```bash
date
```

Output:

```
Fri Nov  3 00:30:05 IST 2023
```

* Get only date part:

```bash
date -I
```

Output:

```
2023-11-03
```

* Get current hour:

```bash
date +%H
```

Output:

```
00
```

* Get current minute:

```bash
date +%M
```

Output:

```
30
```

* Get current second:

```bash
date +%S
```

Output:

```
05
```

* Get current time in Unix seconds

```bash
date +%s
```

Output:

```
1698951605
```
